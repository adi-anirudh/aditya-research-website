# Aditya Anirudh Jonnalagadda — Personal Research Website

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Push this folder to GitHub and import the repository into Vercel.
Vercel should auto-detect Vite. Use:

- Build command: `npm run build`
- Output directory: `dist`
